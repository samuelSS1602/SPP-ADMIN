# Sri Padmavati Pleasant - Quick Start Guide

## 🚀 Start Using in 30 Seconds

### Step 1: Download Files
Download these 3 files:
- `index.html`
- `styles.css`
- `script.js`

### Step 2: Place Files Together
Put all 3 files in the same folder (e.g., create a folder named "HotelSystem")

### Step 3: Open in Browser
- **Windows**: Double-click `index.html`
- **Mac**: Right-click `index.html` → Open With → Chrome/Safari
- **Linux**: Open `index.html` with your browser

### Done! ✅
Your hotel management system is ready to use!

## 💾 Your Data

**Where is data saved?**
- In your browser's memory
- Automatically saves to local storage
- When you close browser and reopen, data is still there

**Reset/Clear data:**
- Press Ctrl+Shift+Delete → Clear browsing data
- Or use browser settings

## 🚨 If Something Goes Wrong

### Webcam not working:
1. Check browser has camera permission
2. Try different browser
3. Restart browser
4. Check camera isn't used elsewhere

### Files not loading:
1. Make sure all 3 files in same folder
2. Clear browser cache (Ctrl+Shift+Delete)
3. Try different browser
4. Check filename spelling

### Data disappeared:
1. Browser cache was cleared
2. Check if in private/incognito mode
3. Try different browser

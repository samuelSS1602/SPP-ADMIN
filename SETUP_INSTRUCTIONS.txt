## 🆘 Troubleshooting

### Problem: File not found / won't open
**Solution:**
- Check all files are in same folder
- Check filename spelling (case-sensitive on Mac/Linux)
- Try opening with different browser
- Check file permissions (might need admin rights)

### Problem: Styling looks weird / colors wrong
**Solution:**
- Refresh browser (F5)
- Clear browser cache (Ctrl+Shift+Delete)
- Try different browser
- Check internet connection (if using online fonts)

### Problem: Webcam doesn't work
**Solution:**
- Click "Allow" when browser asks for camera permission
- Check camera isn't used by another app
- Try different browser (Chrome is most reliable)
- Restart browser
- Check camera device is working

### Problem: Data disappeared
**Solution:**
- Data is stored locally, not deleted
- Try opening in different browser
- Check you're not in private/incognito mode
- Clear cache and reload

### Problem: Features don't respond / nothing happens
**Solution:**
- Check JavaScript is enabled in browser
- Refresh page (F5)
- Check browser console for errors (F12)
- Update browser to latest version
- Try different browser
